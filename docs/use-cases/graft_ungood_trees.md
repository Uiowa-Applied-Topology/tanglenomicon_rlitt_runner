# Graft Ungood Trees

## Primary actor

N/A

## Trigger

An upstream actor wants to graft the zero tangle and a page of scions.

## Goal

Produce a collection of ungood grafted tangle trees.

## Preconditions

C libraries have been built into a cython module.

## Scenario

1. An actor initiates a use case which grafts ungood tangles.
2. A collection of scions is grafted to the zero tangle.
3. New tangles are reported.
