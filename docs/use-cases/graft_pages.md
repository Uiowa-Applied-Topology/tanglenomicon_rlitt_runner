# Graft Pages

## Primary actor

N/A

## Trigger

An upstream actor wants to graft a page of rootstocks and a page of scions.

## Goal

Produce a collection of grafted tangle trees.

## Preconditions

C libraries have been built into a cython module.

## Scenario

1. An actor initiates a use case which grafts tangles.
2. A collection of scions is grafted to a collection of rootstocks.
3. New tangles are reported.
