# Retrieve Page

## Primary actor

N/A

## Trigger

An upstream actor wants to retrieve a page from the tangle collection.

## Goal

Retrieve a page of tangles from long term storage.

## Preconditions

Long term storage is attached.

## Scenario

1. An actor initiates a use case which requires a page of tangles.
2. Long term storage is parsed for a given page of tangles.
3. The page is reported.
